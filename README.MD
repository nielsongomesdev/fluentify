# 🌐 Fluentify - Language Course Landing Page

Uma Landing Page responsiva e bilíngue desenvolvida como projeto acadêmico para aplicar fundamentos de desenvolvimento Front-End. O projeto simula a página de captura de um curso de idiomas moderno, com foco na experiência do usuário e validação de dados.

## Funcionalidades

* **Design Responsivo:** Interface construída com **Bootstrap 5**, adaptando-se perfeitamente a celulares, tablets e computadores.
* **Sistema Bilíngue (i18n):** Suporte para troca instantânea de idioma (Inglês e Português) em toda a página utilizando Vanilla JavaScript, sem recarregar a tela.
* **Validação de Formulário:** Verificação avançada de campos (nome, e-mail via Regex e tamanho da mensagem) com feedback visual em tempo real.
* **Redirecionamento:** Encaminhamento dinâmico para uma página de agradecimento (`obrigado.html`) após o preenchimento correto do formulário.
* **Estrutura Semântica:** HTML5 organizado com `header`, `nav`, `section`, `form` e `footer`, atendendo aos padrões de acessibilidade e SEO.

## 🛠️ Tecnologias Utilizadas

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla/ES6+)**
* **Bootstrap 5**
